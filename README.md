# 树洞弹幕前端项目

## 技术栈
Vue 3 + Vite

## 项目运行
```
npm install
npm run dev
```
配合后端项目一起使用，修改src/http/axios.js中的baseURL为后端项目的地址。

## 项目结构
```
├── public
│   ├── index.html
├── src
│   ├── assets
│   ├── components
│   ├── router
│   ├── store
│   ├── styles
│   ├── utils
│   ├── views
│   ├── App.vue
│   ├── main.js
│   └── vite.config.js
├── .gitignore
├──  babel.config.js
├──  package.json
├──  README.md
└──  vite.config.ts
```
